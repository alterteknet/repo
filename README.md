# JumpCloud Agent Installation Script

## Deskripsi
Repositori ini berisi **script Bash** untuk melakukan instalasi **JumpCloud Agent** secara otomatis pada sistem Linux (Debian/Ubuntu-based), termasuk proses update repository, instalasi dependensi, serta eksekusi Kickstart resmi JumpCloud.

Script ini dikembangkan **khusus untuk kebutuhan internal proyek** dan **bukan untuk distribusi publik**.
